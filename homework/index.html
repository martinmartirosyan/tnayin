Ո՞րն է տարբերությունը տեղական և հեռավոր ֆայլերի միջև Linux-ում:

Տարբեր սերվերների երկու ֆայլերի միջև տարբերությունը համեմատելու կամ գտնելու
համար գործարկեք հետևյալ հրամանը. Հիշեք, որ օգտագործողին և հեռակա հոսթին փոխարինեք
ձեր իրական պարամետրերով: Այնուհետև օգտագործեք cat հրամանը՝ diff_output-ի բովանդակությունը դիտելու համար: txt ֆայլ:

Ինչպե՞ս կարող եմ վերահիմնավորել տեղական մակարդակում:


Ձեր ընթացիկ տեղական մասնաճյուղում ավելացրեք ձեր բոլոր ընթացիկ փոփոխությունները, կատարեք և մղեք
դրանք սկզբնավորմանը: git ավելացնել. Վերադարձեք ձեր տեղական(local) մասնաճյուղ և վերանայեք վարպետը:
Այնուհետև ուժով մղել (push)փոփոխությունները դեպի ծագում:

Ինչպե՞ս է Git-ն աշխատում հեռավոր պահեստի հետ:

Git-ում կան ընդամենը մի քանի հրամաններ, որոնք փոխազդում են հեռավոր պահեստի հետ:
Աշխատանքի ճնշող մեծամասնությունը տեղի է ունենում տեղական պահեստում: Մինչ այս պահը
(բացառությամբ այն դեպքերի, երբ մենք անվանում էինք «git clone»), մենք աշխատել ենք բացառապես
մեր տեղական Git պահեստի հետ և երբեք չենք լքել մեր տեղական համակարգիչը:

Ո՞րն է տարբերությունը հեռավոր պահեստի և տեղական պահեստի միջև:

Հեռավոր պահեստի (օրինակ՝ GitHub) նպատակն է հրապարակել ձեր կոդը աշխարհին
(կամ որոշ մարդկանց) և թույլ տալ նրանց կարդալ կամ գրել այն: Հեռավոր պահոցը
ներգրավված է միայն այն դեպքում, երբ դուք git-ը մղում եք ձեր տեղական
պարտավորությունները դեպի հեռավոր պահոց, կամ երբ դուք git-ը հանում եք ուրիշի պարտավորությունները դրանից:

Որո՞նք են Git Repos-ի տարբեր տեսակները:


Git-ն ունի պահեստի երկու տեսակ՝ տեղական և հեռավոր: Տեղական ռեպո-ն ձեր համակարգչում է
միայն ձեր անմիջական օգտագործման համար: Հեռավոր ռեպո-ն սովորաբար և ձեր օգտագործման համար:
Git-ն աջակցում է բազմաթիվ հեռավոր պահոցներ: Սովորաբար մենք աշխատում ենք թիմերով և պետք է միասին աշխատենք կոդի բազայի վրա:

Ինչպե՞ս պաշտոնականացնել git տեղական պահեստի սահմանումը:

Այս «տեղական պահոցը» հետևում է աշխատանքային գրացուցակի ֆայլերին, բեմադրության տարածքին/ինդեքսին,
որտեղ ֆայլերը ավելացվում են, և HEAD-ը, որտեղ այն մատնանշում է ամենավերջին կատարման մասին:
Ինչպե՞ս կարող եմ պաշտոնականացնել տեղական պահեստի սահմանումը: Հավանաբար մեկ այլ հարցի այս պատասխանը կարող է օգնել ձեզ.
